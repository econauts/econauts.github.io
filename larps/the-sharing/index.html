The Sharing

The Sharing is a sci-fi secrets-and-powers LARP set in the world of K.A. Applegate’s Animorphs. This is a game of secrecy and paranoia, where you can never be sure who to trust and who has an evil mind-controlling slug lurking behind their friendly grin.

Those who fondly remember the Animorphs books being about fun animal-morphing adventures and wacky alien hijinks, be advised: the series gets quite dark, and this game investigates many of the darker aspects, such as brainwashing, ableism, and child neglect.

This is a game for fifteen players, runs for four hours, and can be run as a print-and-play, though props, music, lighting and scenography can add to a sense of atmosphere. The game contains American and European localizations. You can download the game as a .zip file.
